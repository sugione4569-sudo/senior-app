프로젝트: 알리의 마법 지팡이 (시니어 바이브 코딩 가이드)

1. 프로젝트 개요

목표: 시니어 사용자가 '말 한마디'로 화면을 바꾸는 체험을 통해 '바이브 코딩'의 개념을 쉽게 이해하도록 도움.

주요 페르소나: 60~80대 시니어 (큰 글씨, 직관적 UI, 따뜻한 안내 필요)

핵심 기술: React, Tailwind CSS, Web Speech API (STT/TTS)

2. 개발 단계 (Cursor 지시 가이드)

1단계: 기본 환경 및 UI 구성

App.jsx 파일 하나에 모든 로직을 통합하여 관리한다.

배경은 짙은 남색 오로라 애니메이션을 적용한다.

중앙에 커다란 황금색 마이크 버튼을 배치한다.

2단계: 캐릭터 및 오디오 엔진

캐릭터: SVG 기반의 꼬마 로봇 '알리'를 구현 (이미지 경로 문제 방지).

음성 인식(STT): 브라우저 내장 window.webkitSpeechRecognition을 사용하여 한국어 명령어를 인식한다.

음성 출력(TTS): window.speechSynthesis를 사용하여 알리의 목소리를 구현한다 (속도는 0.85배).

3단계: 색깔 마법 로직 구현

사용자가 "빨갛게", "파랗게", "노랗게"라고 말하면 다음이 동시에 작동해야 함:

배경색이 해당 색으로 부드럽게 전환.

마이크 주변에서 반짝이는 파티클(별가루)이 터져 나감.

알리가 칭찬 멘트를 하며 위아래로 뜀.

3. Cursor에게 줄 핵심 프롬프트

"첨부한 roadmap.md 파일의 내용을 바탕으로, 시니어들을 위한 교육 앱의 첫 화면 프로토타입을 React로 작성해줘. 접근성을 위해 글자 크기는 24px 이상으로 하고, 모든 버튼과 텍스트는 한국어로 작성해. 특히 마이크를 눌렀을 때 음성으로 색깔을 바꾸는 '색깔 마법' 기능이 가장 중요해."
